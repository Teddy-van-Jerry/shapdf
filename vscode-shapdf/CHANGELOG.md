# Change Log

All notable changes to the "shapdf" extension will be documented in this file.

## [0.2.1] - 2025-11-04

### Added
- Initial release
- Syntax highlighting for `.shapdf` files
- PDF preview using shapdf CLI
- Auto-preview on save (configurable)
- Side-by-side preview with keyboard shortcut (`Ctrl+Shift+V` / `Cmd+Shift+V`)
- Configuration options for CLI path and auto-preview behavior
